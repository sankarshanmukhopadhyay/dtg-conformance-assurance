# Governance

This repo supports DTG WG review and iterative refinement.

## Roles
- **Maintainers:** manage PR merges, labels, and releases
- **Contributors:** propose changes via Issues/PRs
- **Reviewers:** comment, request changes, and approve PRs

## Change process
1. Open an Issue (risk / control / profile / editorial)
2. Submit a PR referencing the Issue
3. Ensure exports/validation pass
4. Maintainer merges to `main`
